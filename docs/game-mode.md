# 게임모드 (로그라이크 던전) 기획서

## 개요
- 기존 전투모드와 **별도 메뉴**로 신규 추가
- 무한 방 진행, 죽으면 처음부터 (로그라이크)
- 블랭크 드릴 기반 실시간 턴제 전투
- 런 내 성장 + 런 간 영구 성장

---

## 메뉴 구조 변경
- 기존 "전투" 버튼 → **"보스"** (이름만 변경, 코드 유지)
- **"게임"** 버튼 신규 추가
- 게임 진입 전 화면: 영구 업그레이드, 시작 버프 선택, 최고 기록 확인

---

## 전투 시스템

### 블랭크 드릴 전투
- 구절에 블랭크(빈칸)가 생성됨
- 한 턴에 블랭크 **1개씩** 엔터로 입력
- 블랭크 순서 무관, 맞는 답이면 해당 블랭크에 표시
- **모든 블랭크를 맞추면** → 플레이어 공격
- **오답 또는 시간초과** → 몬스터 공격 (플레이어 피격)

### 블랭크 수 (몬스터 레벨 - 내 레벨 기준)
| 레벨 차이 | 블랭크 수 |
|-----------|-----------|
| -1 이하 (나보다 낮음) | 1개 |
| 0 (동급) | 2개 |
| +1 | 3개 |
| +2 | 4개 |
| +3 이상 | 5개 |

### 제한 시간
- **주관식 모드 (recite_mode 0)**: 블랭크 1개당 10초
- **빈칸채우기 모드 (recite_mode 1)**: 시간제한 없음
- **레벨 차이 5 이상**: 블랭크 1개당 2초 (모드 무관, 극한 난이도)

### 구절 범위
- 관리자가 설정한 전체 말씀 구절이 베이스
- 방 1~5: 1절만
- 방 6~8: 1~2절
- 방 9~11: 1~3절
- 이후 **3방마다 +1절** 증가

---

## 몬스터 시스템

### 몬스터 레벨 (방 깊이 기반)
- 방 1: 레벨 0~5
- 방 2: 레벨 4~10
- 이후 방마다 레벨 범위가 점진적 상승 (무한)
- 플레이어 레벨과 무관하게 방 깊이로 결정

### 게임모드 전용 몬스터 스탯 (3~5턴 킬 밸런스)
- 기존 전투모드 스탯과 별도
- HP, 공격력, 방어력을 축소하여 빠른 전투 지향

### 몬스터 외형
- 방 깊이에 따라 마을 순서대로 몬스터 풀 사용
  - 방 1~N: 마을 1 몬스터 (에덴)
  - 방 N+1~M: 마을 2 몬스터 (기쁨의 동산)
  - ...순차 진행
- 각 마을 풀 내에서 랜덤 선택, 중복 출현 가능

### 난이도 상승
- 3마리마다 난이도 단계 상승 (변수화, 추후 조정 가능)
- 난이도 상승 상한 없음 (무한)

---

## 데미지 공식

### 플레이어 공격 (모든 블랭크 정답 시)
- 기본: `max(minDmg, 내 공격력 - 몬스터 방어력)`
- 최종: minDmg ~ maxDmg 범위 내 랜덤
- 기존 전투 공식 기반, min/max 범위 적용

### 몬스터 공격 (오답/시간초과 시)
- 기본: `max(minDmg, 몬스터 공격력 - 내 방어력)`
- 최종: minDmg ~ maxDmg 범위 내 랜덤

---

## 방 진행 구조

### 갈림길
- 매 방마다 좌/우 선택 (시각적 재미, 결과는 랜덤)
- 뒤로 갈 수 없음

### 상점 (5방마다)
| 항목 | 효과 | 가격 |
|------|------|------|
| HP 회복 | 최대 HP의 30% 회복 | 금화 3 |
| 공격력 강화 | 공격력 +15% (런 한정) | 금화 5 |
| 방어력 강화 | 방어력 +15% (런 한정) | 금화 5 |
| HP 최대치 증가 | 최대 HP +20% (런 한정) | 금화 7 |

- 가격은 방 깊이와 무관하게 고정

### 엘리트 몬스터 (10방마다)
- 스탯: 일반 몬스터의 **HP 1.5배, 공격력 1.3배**
- 선택: "도전" vs "도망"
  - **도전 승리**: 아이템 100% 드랍 + 금화 2배 + 별(영구재화) 보너스
  - **도전 패배**: 일반 사망 처리
  - **도망**: 패널티 없이 다음 방 진행
- 외형: 이름 앞에 "⚡", 빨간 테두리
- **상점과 겹칠 때 (방 10, 20, 30...)**: 엘리트 먼저 → 승리 시 상점 등장

### 랜덤 이벤트 (확률 15%)
- 상점/엘리트가 아닌 방에서 발생
| 이벤트 | 선택지 | 결과 |
|--------|--------|------|
| 보물상자 | 열기 / 무시 | 열기: 70% 아이템, 30% 함정(HP -15%) |
| 상처입은 순례자 | 돕기(금화3) / 지나침 | 돕기: 랜덤 버프 1개 |
| 기도의 샘 | 기도 / 지나침 | 기도: HP 50% 회복, 공격력 -5% (런 한정) |
| 수상한 상인 | 거래(HP 20%) / 거절 | 거래: 랜덤 고급 버프 |
| 고대 석판 | 해독 도전 / 지나침 | 성공: 별 보너스, 실패: 없음 |

## 보상 시스템

### 금화 (런 내 한정)
- 몬스터 처치 시 **고정 2개** 드랍
- 런 종료 시 리셋 (0으로)
- 용도: 상점 구매, 이벤트 비용

### 아이템 드랍
- 드랍률: **40%**
- 장비 60% / 소모품 40%
- 아이템 레벨: 캐릭터 레벨 기준
- **장비는 영구 획득** (런 끝나도 인벤토리 유지)

### EXP
- 몬스터 처치 시 경험치 획득
- **레벨 차이에 따른 감소**:
  - 몬스터가 내 레벨보다 5 이상 낮음: EXP 0
  - 3~4 낮음: EXP 50%
  - 동급 이상: EXP 100%

---

## 콤보 시스템
- 연속 정답 시 콤보 카운터 증가
- 데미지 배율: `1 + 콤보 × 0.05` (콤보 10 = 1.5배)
- **최대 1.5배** (10콤보 캡)
- 오답/시간초과 시 콤보 리셋
- UI: 화면에 콤보 카운터 표시

---

## 런 내 버프 시스템
- **3방마다** 버프 선택 등장 (변수화, 추후 5방/7방 조정 가능)
- 3개 중 1개 선택
- **중복 선택 불가** (같은 버프 재등장 안 함)

### 버프 풀
| 버프 | 효과 |
|------|------|
| 축복의 검 | 공격력 +10% |
| 축복의 방패 | 방어력 +10% |
| 생명의 은총 | 최대 HP +15% |
| 시간의 여유 | 블랭크 제한시간 +3초 |
| 치유의 손길 | HP 20% 즉시 회복 |
| 전사의 눈 | 크리티컬 확률 +10% |
| 상인의 축복 | 금화 드랍 +1 |
| 연속 기도 | 콤보 데미지 배율 +5% |

---

## 영구 성장 시스템

### 별 (⭐ 영구 재화)
- 런 종료 시 **도달한 방 수 = 획득 별 수**
- 엘리트 처치 시 추가 별 보너스

### 영구 업그레이드
| 업그레이드 | 1단계 | 2단계 | 3단계 | 4단계 | 5단계 |
|-----------|-------|-------|-------|-------|-------|
| 기본 HP 강화 (+5%씩) | ⭐10 | ⭐25 | ⭐50 | ⭐80 | ⭐120 |
| 기본 공격력 강화 (+5%씩) | ⭐10 | ⭐25 | ⭐50 | ⭐80 | ⭐120 |
| 금화 수집가 (+1씩) | ⭐15 | ⭐40 | ⭐75 | - | - |
| 상인의 친구 (-10%씩) | ⭐15 | ⭐40 | ⭐75 | - | - |
| 축복의 기운 (선택지+1) | ⭐20 | ⭐50 | ⭐90 | - | - |
| 부활의 기도 (1회 부활) | ⭐100 | - | - | - | - |

## 해금 콘텐츠
| 해금 조건 | 보상 |
|-----------|------|
| 방 10 달성 | 시작 버프 "전사의 축복" (공격력 +10%) 선택 가능 |
| 방 20 달성 | 시작 버프 "수호자의 축복" (방어력 +15%) 선택 가능 |
| 방 30 달성 | 시작 버프 "생명의 축복" (HP +20%) 선택 가능 |
| 업적 5개 달성 | 시작 버프 슬롯 2개 (동시 선택) |

- 시작 버프는 기본 1개만 선택 가능 (슬롯 해금 전까지)

## HP / 생존
- 방 사이에 HP 유지 (로그라이크)
- HP 0 = 사망 → 최고 방 기록 저장, 별 지급, 처음부터 재시작
- 자발적 퇴장 = HP 0 사망 처리 (기록 저장)
- 회복 수단: 상점(30%), 버프(20%), 이벤트(기도의 샘 50%)

---

## 기존 소모품
- 게임모드에서는 **소모품 미적용** (추후 논의)

---

## 기술 참고
- 기존 전투 코드(MonsterEncounter, BattleScreen, battle.ts)는 **수정하지 않음**
- 게임모드 전용 컴포넌트/핸들러를 신규 생성
- 설정 변수화: 버프 주기(3방), 상점 주기(5방), 엘리트 주기(10방) 등
