# 스토리 기획서: "말씀의 빛, 잃어버린 세계"

## 세계관

태초에 하나님의 **말씀의 빛**이 온 세상을 비추고 있었다. 그런데 어둠의 세력이 말씀의 빛을 **20개의 조각**으로 부수어 세상 곳곳에 흩어버렸다. 빛을 잃은 세상은 어둠에 잠식되어 몬스터들이 들끓게 되었고, 사람들은 마을에 갇혀 살아간다.

플레이어는 **말씀의 전사** — 성경 말씀을 암송하는 힘으로 어둠을 물리치고, 각 마을에 흩어진 **말씀의 조각**을 되찾아 세상에 빛을 회복시키는 여정을 떠난다.

> 그러나 어둠의 세력이 말씀의 빛을 산산이 부수어 스무 조각으로 흩어버리니, 세상은 어둠에 잠식되고 짐승들이 들끓기 시작하였더라.

**[화면이 어둡게 변하며 몬스터 그림자들이 스쳐 지나간다]**

> 사람들은 마을에 갇혀 두려움 속에 살아갔다. 그때, 에덴의 장로가 예언하였으니—

**[에덴 마을, 장로 NPC 등장]**

장로:
> "말씀을 마음에 새기는 자가 나타나리라. 그가 어둠을 물리치고 빛의 조각을 되찾으리니..."

**[플레이어 캐릭터가 화면에 등장 — 선택한 아바타 + 닉네임 표시]**

> "너는 여느 전사와 다르다. 칼과 방패가 아닌, **말씀의 힘**으로 싸우는 자이니라."

**[장로가 빛나는 두루마리를 건네는 연출]**

장로:

> "에덴 바깥은 이미 어둠에 물들었다. 하지만 두려워 말라. 말씀을 네 마음에 새기면 어떤 어둠도 너를 이기지 못하리라."

**[선택지 등장]**

{닉네임}:
> **"제가 할 수 있을까요?"**

장로:
> "할 수 있느냐고? 하하... **{닉네임}**, 네가 하는 것이 아니다. 말씀이 하는 것이니라. 네가 말씀을 외우면, 말씀이 너를 통해 싸울 것이다."

> "자, 먼저 이 에덴의 들짐승들부터 상대해보거라. 그것이 네 첫 번째 시험이니라."

**[화면 전환 — 메인 화면으로]**

*시스템 메시지:*
> 📜 **말씀의 전사 {닉네임}의 여정이 시작됩니다.**
> 🌳 에덴의 몬스터를 물리치고 첫 번째 빛의 조각을 되찾으세요!

**[꽃이 만발한 정원, 하지만 일부 꽃이 시들어 있다]**

꽃지기:
> "어머, 여행자! 아니... 혹시 당신이 그 말씀의 전사 **{닉네임}**?"

> "이 동산을 좀 봐요. 예전엔 온통 꽃이었는데, 어둠이 스며든 뒤로 벌레들이 꽃을 갉아먹기 시작했어요."

> "빛의 조각이 사라지자 곤충들이 거대해지고 흉폭해졌죠. 부탁이에요, 이곳의 빛을 되찾아주세요."

{닉네임}:
> **"꽃이 다시 필 수 있을까요?"**

꽃지기:

### 마을 3. 속삭임의 숲 🌲 — 진입

**[어두운 숲, 나무 사이로 그림자가 속삭인다]**

*속삭임:*
> *"돌아가라... 여긴 네가 올 곳이 아니다..."*
> *"말씀 따위 잊어버려라... 편하게 살 수 있다..."*

숲의 파수꾼:
> "...! **{닉네임}**, 귀를 막아라! 이 숲의 그림자들은 유혹의 속삭임으로 전사들의 마음을 무너뜨린다."

> "여기서 쓰러진 전사가 한둘이 아니다. 모두 '조금만 쉬자'는 속삭임에 넘어가 다시는 일어나지 못했지."

> "하지만 네가 말씀을 입에 담고 있는 한, 그 속삭임은 너를 이기지 못한다. 기억하거라 — **말씀이 곧 방패**다."

### 마을 4. 실락원 🌫️ — 진입

**[안개 자욱한 황무지, 부서진 건물 잔해]**

방랑자:
> "이곳이 한때 낙원이었다는 걸 믿겠나, **{닉네임}**?"

> "빛의 조각이 사라지고... 타락한 천사들이 이곳을 지배하기 시작했지. 사람들은 모두 떠났어. 나만 빼고."

{닉네임}:
> **"왜 남아계신 건가요?"**

방랑자:
> "누군가 와서 이곳을 되살려줄 거라 믿었거든. 그리고... 네가 왔구나."

> "조심해라. 여기 타락천사들은 지금까지 상대한 것들과 차원이 다르다. 네 믿음이 진짜인지 시험당할 것이다."

### 마을 5. 가시의 땅 🏜️ — 진입

**[끝없는 사막, 갈라진 대지 위에 가시덤불]**

사막의 순례자:
> "**{닉네임}**... 여기까지 왔다니. 대단한 전사로구나."

> "하지만 여기서부턴 진짜 고난이다. 이 사막은 끝이 없어 보이고, 가시가 살을 찢고, 뜨거운 태양이 의지를 꺾으려 하지."

> "많은 전사들이 이 사막 한가운데서 포기했다. '차라리 에덴으로 돌아가자'고."

> "하지만 기억하거라. 가시밭을 지나지 않으면 꽃밭에 도달할 수 없느니라. **고난이 인내를, 인내가 연단을 만든다.**"

### 마을 6. 바벨의 파편 🏛️ — 진입

> "이 탑을 쌓은 자들은 말씀이 아닌 자신의 힘을 믿었지. '우리가 곧 신이 되리라!' 하늘을 향해 외쳤으나..."

> "결국 탑은 무너지고, 서로의 말을 알아듣지 못하게 되었다."

> "이곳의 몬스터들도 그렇다. 교만으로 뒤틀린 존재들... 오직 겸손한 자의 말씀만이 이들을 물리칠 수 있느니라."

### 마을 7. 소돔의 잿더미 🔥 — 진입

**[유황불에 탄 폐허, 연기가 피어오른다]**

재 속의 생존자:
> "냄새가 나지, **{닉네임}**? 유황 냄새... 수백 년이 지났는데도 사라지지 않아."

> "이 도시는 심판받았다. 빛의 조각이 경고했건만 아무도 듣지 않았지."

> "한 가지 충고하마. 이곳에서는 **절대 뒤돌아보지 마라.** 뒤를 돌아본 자는 소금기둥이 되어 영원히 이곳에 갇히게 된다."

> "오직 앞만 보고, 말씀만 붙잡아라. 그것이 이 잿더미를 통과하는 유일한 길이다."

> "이 심연의 몬스터들은 물속에서 온다. 너도 방주를 가져야 해. 네 방주는 바로 **네가 외운 말씀**이다."

> "말씀 위에 서 있는 한, 아무리 깊은 물도 너를 삼키지 못하리라."

### 보스전 vs 일반전 차이

| 항목 | 일반전 | 보스전 |
|------|--------|--------|
| **진입 연출** | 바로 전투 | 보스 등장 컷신 (보스 이름 + 대사) |
| **배경** | 마을 기본 배경 | 어둡게 + 붉은/보라 톤 강조 |
| **BGM** | 일반 전투 | 보스 전용 긴장감 있는 BGM |
| **몬스터 표시** | 이모지 + 이름 | 큰 이모지 + 이름 + 칭호 + HP바 |
| **전투 턴** | 3~6턴 | 6~10턴 (더 긴 전투) |
| **승률 보정** | 기본 공식 | 기본 공식 - 10% (더 어려움) |
| **패배 시** | 결과 화면 | 보스 도발 대사 + "다시 도전하시겠습니까?" |
| **승리 시** | 아이템 드랍 | 빛의 조각 획득 컷신 + 보스 전용 보상 확정 |

### 보스 등장 연출 (공통 포맷)

```
1. 화면 전체 어둡게 페이드 (0.5초)
2. 화면 흔들림 효과 (0.3초 x 3회)
3. 붉은/보라 빛 번쩍 효과
4. 보스 이모지가 화면 중앙에 크게 등장 (2~3배 확대 애니메이션)
5. 칭호 + 보스 이름 텍스트 등장 (예: "⚔️ 에덴의 지배자 — 타락한 뱀 🐍")
6. 보스 등장 대사 표시
7. "전투 시작" 버튼
```

#### 일반 몬스터 vs 보스 등장 차이

| 항목 | 일반 몬스터 | 보스 |
|------|-----------|------|
| 등장 | 바로 표시 | 암전 → 흔들림 → 빛 번쩍 → 등장 |
| 이모지 크기 | 보통 | 2~3배 크게 |
| 이름 표시 | 이름만 | 칭호 + 이름 |
| 배경 | 마을 기본 | 어둡게 + 붉은/보라 톤 |
| 대사 | 없음 | 등장 대사 + 처치 후 대사 |
| HP 표시 | 없음 | HP바 표시 (실시간 감소) |
| 효과음/BGM | 일반 | 긴장감 있는 보스 전용 |

구현: CSS 애니메이션 (keyframes, transform, filter 등)

### 보스전 전투 방식

#### 기본 구조
- PvP 모드와 유사한 **실시간 암송 대결** 방식
- **암송 제출 없이** 바로 보스전 돌입 (일반 암송 화면 거치지 않음)
- 해당 마을의 누적 출제 범위에서 문제 출제
  - 1절만 배정된 마을: 같은 절에서 빈칸 위치를 매번 변경하며 반복
  - 2절 이상 배정된 마을: 누적 범위 내 구절을 **랜덤** 출제
- **정답** → 플레이어가 보스에게 [플레이어 공격력]만큼 데미지
- **오답/시간초과** → 보스가 플레이어에게 [보스 공격력 - 플레이어 방어력]만큼 데미지 (회피 판정 먼저)
- 보스 HP ≤ 0 → 승리 / 플레이어 HP ≤ 0 → 패배

#### 경험치
- 보스전에서 **경험치 미지급** (암송 EXP, 전투 EXP 모두 없음)
- 보스전의 목적은 "이미 외운 것을 증명하는 시험"
- 레벨 뻥튀기 악용 방지

#### 진입 시점
- 다음 마을 해금 레벨 도달 후, **암송 버튼 클릭 시 일반 암송 화면 대신 바로 보스 컷신 → 보스전 돌입**
- 보스 미클리어 시 해당 마을에서 암송 버튼을 누를 때마다 보스전 돌입 (일반 암송 불가)

#### 패배 후 흐름
- 캐릭터 화면으로 복귀
- 사용자가 원하면 연속 재도전 가능

#### 제한시간
- **빈칸 모드**: 40초
- **주관식 모드**: 15초
- 시간초과 = 오답 처리

#### 최소 라운드 보장
- **보스 HP = 플레이어 공격력 × 10**
- 전부 정답이어도 최소 10문제는 풀어야 클리어
- 오답이 나올수록 라운드 수 증가

#### 플레이어 스탯 (장비 반영)
- **HP**: 기본 HP + 장비 HP 합산 → 실수 허용 횟수 결정
- **공격력**: 장비 공격력 합산 → 보스 HP에 비례하므로 클리어 속도엔 영향 없음
- **방어력**: 장비 방어력 합산 → 오답 시 받는 데미지 감소
- **회피율**: 장비 회피 합산 → 오답이어도 피격 회피 확률

#### 장비 등급별 체감 차이
| 장비 등급 | 실수 허용 횟수 | 체감 |
|----------|--------------|------|
| 장비 없음 | 1~2번 | 거의 불가능 — 장비 파밍 필요 |
| 일반 풀셋 | 2~3번 | 빡빡하지만 가능 |
| 고급 풀셋 | 3~4번 | 적당한 난이도 |
| 희귀 풀셋 | 4~5번 | 여유 있음 |
| 전설 풀셋 | 5~6번 | 넉넉함 |
| 신화 풀셋 | 7번 이상 | 압도적 — 방어력이 공격력 초과 가능 |

#### 소모품
- 보스전에서 소모품(암송 100점권, 힌트권) **사용 불가**
- 순수 실력 + 장비 기반 전투

#### 중간 이탈
- 앱 종료/나가기 시 **처음부터 다시**

#### 구절 미등록 시
- 보스전에 필요한 절이 DB에 없으면 진입 차단
- 안내 메시지: "아직 준비되지 않은 전투입니다"

## 연출 방식

- **배경 이미지 + 텍스트 오버레이** (다음 버튼으로 대사 진행)
- **캐릭터 아바타**가 대화창 옆에 표시
- **NPC 초상화**는 이모지 또는 간단한 아이콘으로 표현
- **첫 진입 시 1회만** 재생, 이후 자동 스킵
- **스킵 불가** — 다음 버튼으로만 진행
- 설정에서 **다시 보기** 가능

## 빛의 조각 UI

- **캐릭터 정보 화면**에 빛의 조각 진행도 표시 (0/20 ~ 20/20)
- 마을 선택 화면에서는 별도 표시 없음 (보스 미클리어 시 다음 마을 잠금으로 충분)
